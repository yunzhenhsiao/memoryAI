# 開發與更新日誌 (Changelog)

這份文件紀錄了 MemoryAI 從專案建立到現在的所有重大版本迭代與功能演進，對應著 Git 上的提交歷史。

---

## 📅 最新更新 (連續劇大腦與跨國時區)

### `Uncommitted` | 全局連續劇脈絡 (Global Rolling Context) & 本月劇情回顧

- **跨事件記憶連結**：為了讓 AI 看懂您的日記「連續劇」，資料庫新增 `user_contexts` 表格永久追蹤您的「人生前情提要」。
- **歸檔全面升級**：無論是「網頁批次匯入」、「對話歸檔」還是「本地 Python 大量匯入」，AI 現在都會先讀取前情提要，寫完新回憶後再把新的發展補充進前情提要中，真正實現「無縫接軌」的懂你！
- **本月故事回顧**：在大腦儀表板 (Dashboard) 新增了專屬區塊，AI 會將您整個月的所有記憶片段，撰寫成一篇充滿情感的第一人稱故事小結。
- **背景防呆防斷**：`import_history_v2.py` 加入自動等待防 429 機制，並處理了 Windows Unicode Emoji 列印問題。

### `Uncommitted` | 跨國時區支援 (Timezone Support)

- **AI 自動捕捉時區**：現在當您在對話或日記中提到「紐西蘭時間」或位於國外，AI 會自動捕捉並填入標準時區 (如 `Pacific/Auckland`)，沒提則預設 `Asia/Taipei`。
- **時間軸視覺**：記憶時光機會在帶有國外時區的時間旁顯示小地球標籤 (🌍)，讓您的出國日記一目了然。

## 📅 近期優化與莫蘭迪風格

### `5c41a5d` - `6aef290` | 全面視覺升級與防呆修復

- **全面引入莫蘭迪冷調配色 (Morandi Cool Tone)**：重新設計 `--color-m-*` 系列 CSS 變數，替換原本的預設 Tailwind 顏色。
- **深色/亮色模式系統 (Dark/Light Mode)**：實作無縫切換主題功能，並將偏好記錄在 localStorage。
- **客製化圖示系統**：新增 `Icons.tsx` 來繪製專屬的 SVG 漸層圖示（大腦、軌道、雲朵等）。
- **星系圖強制夜空模式**：就算切換至亮色模式，`MemoryGraph` 也會維持深邃的暗色背景與發光節點效果。
- **修復渲染當機 (React Render Crash)**：修復在「記憶時光機」頁面切換主題時，因部分缺乏關鍵字資料而呼叫 `.toLowerCase()` 導致白畫面當機的問題。
- **批次匯入介面升級**：將 `BatchImport` 頁面的色系同步套用莫蘭迪設計系統。

### `513736a` | UX 優化 (行動端與連線偵測)

- **行動裝置 RWD 完美適配**：手機小螢幕下時光機中的「修改」與「刪除」按鈕不再消失；隱藏桌面版導覽列的「記憶對話」，將其專屬保留給手機端漢堡選單。
- **多行輸入框**：聊天輸入區塊支援多行 (`textarea`) 自動換行，提升長篇日記輸入體驗。
- **斷線提示**：新增「後端未連接」時的精美彈窗提醒機制。

### `cf3e7d4` - `80797d9` | 原始文本紀錄與批次匯入

- **原始日記文本支援**：記憶時光機現在能夠顯示並編輯經過 AI 處理的摘要，以及「原始日記文本」。並完善了加密文本的後端解密與前端傳輸邏輯。
- **批次匯入優化**：修復日期正則表達式，支援以 YYYY-MM-DD 或 YYYY/MM/DD 自動將大段落日記切割並交由 AI 批次建檔。

## 📅 中期優化 (核心引擎與隱私架構)

### `bb410aa` - `6d73b97` | 大腦星系圖 (Memory Graph)

- **實體節點放大與光暈效果**：將「核心人物/事物」設定為巨大的發光白矮星，加上強大的物理引力排斥設定，與一般記憶節點做出明顯的視覺區隔。

### `76da87c` | 絕對隱私加密防護

- **端到端對稱式加密**：實作應用層等級的 Fernet 加密，所有使用者的對話與事件摘要在寫入資料庫前都會轉換成亂碼，即便是資料庫擁有者也無法窺探。提供專屬的管理員豁免權功能。

### `5120930` - `ef6ad87` | 穩定性修復與 API 優化

- **解決 429 請求過載問題**：針對免費版 Gemini API 常見的 429 (Too Many Requests) 與 503 錯誤，在 Python 後端加入了退避與重試 (Retry) 機制。
- **修復 pgvector 參數**：修正向量庫檢索時的 Cosine Similarity 參數與資料庫 `id` 缺失的 Bug。
